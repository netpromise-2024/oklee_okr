# OK&Lee Family Check-in

가족 6명이 서로의 강점, 바라는 변화, 새로운 도전을 나누고 받은 이야기 중 실천할 항목을 선택해 3개월 동안 실행하는 관계 점검 웹 앱입니다.

## 가족 계정

| 가족 | 이름 | 아이디 |
| --- | --- | --- |
| 아빠 | 옥호성 | `netpromise` |
| 엄마 | 이건화 | `kuna00` |
| 큰아들 | 옥승현 | `hyeon1` |
| 작은아들 | 옥수현 | `hyeon2` |
| 셋째아들 | 옥서현 | `hyeon3` |
| 막내딸 | 옥소현 | `hyeon4` |

현재 가족 공통 비밀번호는 기존과 같은 `hyeon00#`입니다.

## 주요 흐름

1. 나를 제외한 가족 5명에게 `잘하고 있는 것`, `개선해줬으면 하는 것`, `도전해줬으면 하는 것`을 항목별 최대 5개 작성합니다.
2. 각자 로그인하면 자신에게 도착한 이야기만 모아서 볼 수 있습니다.
3. 개선 또는 도전 항목 중 이번 3개월에 실천할 내용을 선택합니다.
4. 선택한 항목에 작은 실행 계획을 적고 진행률을 관리합니다.
5. 부모 계정은 현재 관계 점검을 보관하고 다음 3개월 주기를 시작할 수 있습니다.

## Render 배포

- Type: Static Site
- Build Command: 비워두거나 `echo "no build"`
- Publish Directory: `.`

## Supabase

`supabase/setup.sql`을 Supabase SQL Editor에서 실행합니다. 기존 OKR 테이블은 유지되고, 관계 점검용으로 아래 테이블이 추가됩니다.

- `family_relationship_state`: 현재 3개월 관계 점검
- `family_relationship_cycles`: 지난 관계 점검 보관

이 앱의 로그인은 가족을 구분해 각자 화면을 제공하기 위한 간단한 앱 로그인입니다. 의료·금융 정보처럼 민감한 데이터를 보관하는 용도의 강한 인증 체계는 아닙니다.
